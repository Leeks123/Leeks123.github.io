# 내장 타입

자바스크립트에는 다음 7가지 내장 타입이 있다.

  • null
  • undefined
  • boolean
  • number
  • string
  • object
  • symbol

object를 제외한 나머지는 원시타입이다.

typeof undefined === "undefined"; // true
typeof true === "boolean"; // true
typeof 42 === "number"; // true
typeof "42" === "string"; // true
typeof {} === "object"; // true
typeof Symbol() === "symbol"; // true

typeof 연산자로 원시타입을 확인할 수 있다. 그러나 여기에서 null은 예외이다.

var a = null;

typeof a; // "object"

!a && typeof a === "object"; // true

null은 falsy한 원시값이지만, object이다.


typeof [] === "object"; // true
typeof function() {} === "function"; // true

배열은 숫자 인덱스를 가지고, length 프로퍼티를 가지는 객체이다. 함수의 typeof 반환은 "function"이지만 실제로는 함수도 객체이다.

배열과 함수 모두 객체의 하위 타입이다.

# Pass By Value vs Pass By Reference

값의 타입에 따라 Pass할 방식이 정해진다.

원시값(object를 제외한 내장타입)의 경우 값-복사 방식으로 할당/전달 된다.

let a = 1;
let b = a;

console.log(a, b); // 1 1

b++;

console.log(a, b); // 1 2

객체, 배열, 함수 등의 합성값은 레퍼런스 사본을 생성하여 할당/전달한다.

let a = [1, 2, 3];
let b = a;

console.log(a, b); // [1,2,3] [1,2,3];

b.push(4);

console.log(a, b); // [1,2,3,4] [1,2,3,4]

함수의 인자에서도 마찬가지이다.

function foo(x) {
  x.push(4);
  console.log(x);

  x = [4, 5, 6];
  console.log(x);
}

var a = [1, 2, 3];

foo(a);
// [1,2,3,4]
// [4,5,6]
a;
// [1,2,3,4]

위의 코드에서 a를 인자로 넘기면 a 배열의 레퍼런스가 x에 할당된다.

foo 내부에서 x가 참조하는 배열에 4가 push되고,
이후에 x는 [4,5,6]이 할당되어 더이상 배열 a를 참조할 수 없게 된다.

# 얕은 복사, 깊은 복사

값-복사 방식으로 object 타입의 값들을 사용하려면 레퍼런스가 참조하는 값을 따로 복사하는 과정이 필요하다.

# 얕은 복사

let obj = {
  a: "a",
  b: "b",
};

let clone1 = Object.assign({}, obj);
let clone2 = { ...obj };

console.log(obj, clone1, clone2);
// {a: "a", b: "b"} {a: "a", b: "b"} {a: "a", b: "b"}

clone1.a = "aa";
clone2.b = "bb";

console.log(obj, clone1, clone2);
// {a: "a", b: "b"} {a: "aa", b: "b"} {a: "a", b: "bb"}

위와 같은 코드로 얕게는 복사가 가능하지만, 깊은 복사되지는 않는다.

# 깊은 복사

let obj = {
  a: "a",
  inner: {
    b: "b",
    c: "c",
  },
};
  1. 내부까지 객체들까지 하나하나 복사하기
let deepClone1 = {
  ...obj,
  inner: {
    ...obj.inner,
  },
};
// 이 코드는 귀찮다.
  1. JSON
let deepClone2 = JSON.parse(JSON.stringify(obj));
// 이 코드는 느리다
