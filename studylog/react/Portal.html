# Portal

Portal은 부모 컴포넌트의 DOM 계층 구조 바깥에 있는 DOM 노드로 자식을 렌더링하는 최고의 방법을 제공합니다.

ReactDOM.createPortal(붙일것, 붙는곳);

createPortal을 이용해서 기본적으로 사용하는 #root DOM 노드 말고 다른 DOM 노드를 생성해서 컴포넌트를 그곳에 렌더링 시킬 수 있다.

모달, 팝업 등을 구현하는데 사용하면 굳!

# 간단히 모달 구현

See the Pen React Modal by Leeks123 (@leeks123) on CodePen.

const Modal = ({ children }) => {
  return ReactDOM.createPortal(children, document.getElementById("modal"));
};

const App = () => {
  const [isModalOpen, setIsModalOpen] = React.useState(false);

  const modalOpen = (e) => {
    setIsModalOpen(true);
  };
  const modalClose = (e) => {
    if (
      e.target.className === "modal-dimmed" ||
      e.target.className === "modal-closeBtn"
    ) {
      setIsModalOpen(false);
    }
  };

  return (
    <>
      <button onClick={modalOpen}>click to modal</button>

      {isModalOpen && (
        <Modal>
          <div class="modal-dimmed" onClick={modalClose}>
            <div class="modal-content">
              <h2>Modal</h2>
              <button className="modal-closeBtn" onClick={modalClose}>
                close
              </button>
            </div>
          </div>
        </Modal>
      )}
    </>
  );
};
ReactDOM.render(<App />, document.getElementById("root"));

위의 예제 코드는 HTML파일에 #modal 노드를 만들어 두었다. 이 과정은 동적으로 노드를 만들어도 가능하다. 그에 관한 내용은 Reference를 참조

버튼을 클릭하면 Modal 컴포넌트의 children이 렌더링된다.
